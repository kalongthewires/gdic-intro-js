Conditional Statements

		// OR do this other thing

Objects

Objects let us store a collection of properties.


	var objectName = {
		propertyName: propertyValue, // properties are comma-separated
		propertyName: propertyValue,
		propertyName: propertyValue,
		...
	};
        

	var jenna = {
		hometown: 'Atlanta, GA',
		hair: 'auburn',
		likes: ['knitting', 'code'],
		birthday: {
			month: 10,
			day: 17
		}
	};

Accessing Objects

You can retrieve values using dot notation


	var jenna = {
		hometown: 'Atlanta, GA',
		hair: 'auburn'
	};

	var myHometown = jenna.hometown;

Or using bracket notation (like arrays)


	var myHair = jenna['hair'];

Dot notation is more succinct and readable.

Changing Objects

You can use dot or bracket notation to change properties


  var jenna = {
		hometown: 'Atlanta, GA',
		hair: 'auburn'
	};

	jenna.hair = 'blue';

Add new properties


  jenna.gender = 'female';

Or delete them


  delete jenna.gender;

Array: forEach

An alternative to using a for loop to iterate through an array.

For arrays only!

  for (var counter = 0; counter < lotteryNumbers.length; counter++) {
    document.write(lotteryNumbers[counter] + ' ');

DOM Access

To change something on a page:


  1. Select the DOM node using JS and store it in a variable
    • getElementById()
    • getElementsByClassName()
    • getElementsByTagName()

  2. Use methods to manipulate the node
    • node.innerHTML = '<p>My new HTML</p>';
    • node.textContent = 'My new text';

DOM Access: By ID

You can find nodes by ID using the method:


	document.getElementById(id);

To find:


	<img id="kitten" src="http://placekitten.com/g/200/300" alt="cat"/>

We would use:


	var imgKitten = document.getElementById('kitten');

DOM Access: By Tag Name

You can find nodes that match a certain HTML element type using the method:


  document.getElementsByTagName(tagName);

For example, you could find all paragraphs using:


  document.getElementsByTagName('p');

getElementsByTagName returns an array of nodes.


DOM Access: By Class Name

You can find nodes that match a class using the method:


  document.getElementsByClassName(className);

For example, you could find all the nodes with a class of error using:


  document.getElementsByClassName('error');

getElementsByClassName also returns an array.


DOM Changes: Attributes

You can access and change attributes of DOM nodes using dot notation.


	<img id="kitten" src="http://placekitten.com/g/200/300" alt="cat"/>

We could change the src attribute this way:


	var imgKitten = document.getElementById('kitten');
	var oldSrc = imgKitten.src;
	imgKitten.src = 'http://placekitten.com/g/200/500';

DOM Changes: Styles

You can change element css using style.


  <p class="error" style="display: none;">Something went wrong!</p>

Use this JavaScript:


	var errorMessage = document.getElementsByClassName('error')[0];
	errorMessage.style.display = 'block';

DOM Changes: More Styles

Change any CSS property with a "-" to camelCase, and be sure to include a unit on any number

To achieve the equivalent of this CSS:


	body {
    background-color: pink;
    padding-top: 10px;
  }

Use this JavaScript:


	var pageNode = document.getElementsByTagName('body')[0];
	pageNode.style.backgroundColor = 'pink';
	pageNode.style.paddingTop = '10px';

Events & Listeners

An event is a type of object that is created when the user interacts with a web page.


  <button id="button">Click Me!</button>

To cause a message to appear when the button is clicked, add an event listener to the button:


  var button = document.getElementById('button');

  function clickHandler() {
    alert('You clicked the button!');
  }

  button.addEventListener('click', clickHandler);

Events: addEventListener

  • An event listener triggers a reaction when a specified event occurs.

  • To create an event listener on a specific node, use the format: 
    
  node.addEventListener(eventType, eventHandler);
  • Event handler: a function that is called when the event occurs.

Types of Events

There are a variety of events. Some common events:

  • click: The event occurs when the user clicks on an element
  • mouseover: The event occurs when the pointer is moved onto an element (or its children)
  • mouseout: The event occurs when the pointer is moved off an element (or its children)
  • keypress: The event occurs when the user presses a key on their keyboard
  • load: The event occurs when a document has been loaded

Preventing Default Behavior

Elements like buttons and links have default behaviors. However, the event object has a built-in method to suppress default behavior:


  <a id="contentToggle" href="#">Read More</a>
        

  var contentToggle = document.getElementById('contentToggle');

  function clickHandler(event) {
    event.preventDefault(); // prevents jump to the top of the page
    console.log(event); // what information is in the event object?
  }

  contentToggle.addEventListener('click', clickHandler);

this

The keyword this references the node that the event was triggered from:


  someNode.addEventListener('mouseover', function() {
    console.log(this); // What do we see if we print to the console?
  });
        

  someNode.addEventListener('click', function() {
    this.style.backgroundColor = 'red';
  });

Forms

You can collect information from users to use in functions. The most common method is an HTML form:


  <form id="temperatureForm">
    <label for="temp">Convert temperature:</label>
    <input type="text" id="temp" size="3" placeholder="Enter temperature"/>
    <input type="radio" name="tempFormat" value="F" checked />Fahrenheit
    <input type="radio" name="tempFormat" value="C" />Celsius <br />
    <label for="tempSubmitButton"></label>
    <input id="tempSubmitButton" type="submit" value="Convert" />
  </form>

Retrieving Form Data

You retrieve the values of form elements using the value method:


  var temperature = document.getElementById('temp').value;
  console.log (temperature);

You can retrieve the value of a form at any time. Try listening for the blur event (when a form element loses focus).

Radio Buttons

Radio buttons usually do not have IDs, so you will need to use an array:


var radios = document.getElementsByName('tempFormat');

for (var i = 0, length = radios.length; i < length; i++) {
    if (radios[i].checked) {
        var radioButtonValue = radios[i].value;
        // only one radio can be checked, so stop now
        break;
    }
}

Submit buttons

If you are going to retrieve form values with the submit button, be sure to prevent the default action!


var submitButton = document.getElementById('tempSubmitButton');
submitButton.onclick = function () {
    event.preventDefault();
    var temperature = document.getElementById('temp').value;
    console.log (temperature);
}
